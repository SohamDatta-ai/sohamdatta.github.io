# Soham Datta | AI Researcher Portfolio

This is my personal portfolio and research blog, built using [Quarto](https://quarto.org/). It focuses on emotionally intelligent systems, on-device ML, and human-centered design.

## Features
- **Project Showcase:** Highlights my work on AURA and OS-1.
- **Deep Bio:** My journey from Pune to the frontiers of AI.
- **"Now" Page:** A real-time update of my current focuses and research.
- **Minimalist Aesthetic:** A technical, monochrome design optimized for reading.

## How to Build and Deploy

### Local Development
1. Install [Quarto](https://quarto.org/docs/get-started/).
2. Preview the site:
   ```bash
   quarto preview
   ```

### Deployment to GitHub Pages
The site is configured to render to the `docs/` folder.
1. Render the site:
   ```bash
   quarto render
   ```
2. Commit and push to GitHub:
   ```bash
   git add .
   git commit -m "Update portfolio"
   git push origin main
   ```
3. Ensure your GitHub repo settings have **GitHub Pages** source set to the `docs` folder on the `main` branch.
